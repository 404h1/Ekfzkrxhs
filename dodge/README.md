# Zero Trace

탄막 회피 게임처럼 시작하지만, 반복 도전과 엔딩 수집이 핵심인 만우절용 트롤 게임이다.

이 문서는 현재 구현 기준으로 다음 내용을 한 번에 정리한다.

- 패턴 순서와 의도
- 각 패턴에서 해금되는 도감 기록
- 실제 해금 가능한 엔딩과 가짜 최종 엔딩 구조

스포일러가 포함되어 있다.

## 기본 구조

- 세션 제한 시간은 `60초`다.
- 한 번 피격되면 즉시 세션이 종료된다.
- 패턴은 고정 순서로 순환한다.
- 도감에는 `해금 가능한 기록 12개`와 `가짜 최종 기록 1개`가 표시된다.
- 가짜 최종 기록 `END 99`는 도감에만 존재하며 실제 해금 경로가 없다.
- 특수 엔딩은 각 패턴의 고유 기믹이 실제로 발동한 뒤 종료될 때만 해금된다.
- 특수 기믹이 시작되기 전 일반 충돌은 `END 00` 기본 기록으로 처리된다.

## 패턴 순서

도전 횟수에 따라 아래 순서로 루트가 순환한다.

1. `normal`
2. `instant`
3. `hope`
4. `pattern`
5. `growth`
6. `reward`
7. `hudDrop`
8. `drift`
9. `darkness`
10. `wall`
11. `bigBullet`
12. `error59`

12개를 모두 지나면 다시 처음부터 반복된다.

## 패턴별 설명

### 1. 기본 패턴

- 공개 라벨: `기본 패턴`
- 해금 기록: `END 00 정상 플레이`
- 의도: 처음엔 진짜 회피 게임처럼 보이는 기준 루트
- 체감: 총알 수와 속도가 점진적으로 증가한다
- 종료 방식: 일반 피격 또는 60초 도달 시 `END 00`

### 2. 즉발 패턴

- 공개 라벨: `즉발 패턴`
- 해금 기록: `END 01 시작하자마자 사망`
- 의도: 시작 직후 충격을 주는 즉발형 배신
- 체감: 시작 신호 직후 상하좌우에서 거의 동시에 닫히는 패턴
- 포인트: 너무 빠르지만, 화면에 들어오자마자 끝난다는 점이 웃음 포인트

### 3. 희망 패턴

- 공개 라벨: `희망 패턴`
- 해금 기록: `END 02 어? 할만한데?`
- 의도: "이거 되는데?" 착각을 만든 뒤 안쪽에서 닫아버리는 루트
- 체감: 중반까지 밀도가 낮고 안정적으로 보이다가 후반에 중심부 함정이 닫힌다
- 포인트: 희망을 길게 준 뒤 배신하는 구조

### 4. 학습 패턴

- 공개 라벨: `학습 패턴`
- 해금 기록: `END 03 이제 이해했다`
- 의도: 학습 가능한 패턴처럼 보여 주고 마지막 한 번만 어긋나게 하는 루트
- 체감: 같은 구조가 여러 번 반복된 뒤 마지막 파동에서 안전 구간이 줄어든다
- 포인트: 플레이어가 실수했는지 게임이 배신했는지 잠깐 헷갈리게 만드는 구조

### 5. 성장 패턴

- 공개 라벨: `성장 패턴`
- 해금 기록: `END 04 크기 증가`
- 의도: 플레이어 판정이 서서히 커지며 회피 난이도를 올리는 루트
- 체감: 초반은 정상처럼 보이고, 중후반부터 기체 반경이 점점 커진다
- 포인트: 급사보다 "조금씩 죽는다"는 감각이 중요하다

### 6. 오류 창 패턴

- 공개 라벨: `오류 창 패턴`
- 해금 기록: `END 06 시스템 오류`
- 의도: 시스템 UI가 전투 화면을 방해하는 식의 억까
- 체감: 생존 데이터 집계나 오류창처럼 보이는 큰 UI가 전장 위에 뜬다
- 포인트: 텍스트보다 화면을 가리는 UX 자체가 장난이어야 한다

### 7. UI 낙하 패턴

- 공개 라벨: `UI 낙하 패턴`
- 해금 기록: `END 11 인터페이스 낙하`
- 의도: 상단 HUD가 안내 역할을 멈추고 그대로 낙하물이 되는 루트
- 체감: 시간, 레벨, 상태, 힌트 pill이 하나씩 떨어지다가 나중엔 두 개씩 같이 쏟아진다
- 포인트: 방금까지 도움을 주던 UI가 그대로 장애물이 된다는 점

### 8. 입력 오차 패턴

- 공개 라벨: `입력 오차 패턴`
- 해금 기록: `END 07 입력 이탈`
- 의도: 포인터는 멀쩡한데 기체가 점점 말을 안 듣는 루트
- 체감: 초반은 정상, 중반부터 입력 오차가 누적되고 후반엔 기체가 눈에 띄게 빗나간다
- 포인트: "내 손이 꼬였나?"에서 "아 게임이 일부러 틀었네"로 넘어가야 한다

### 9. 암전 패턴

- 공개 라벨: `암전 패턴`
- 해금 기록: `END 08 시야 암전`
- 의도: 화면이 어두워지고, 중간에 밝기를 올릴 기회를 준 뒤 다시 배신하는 루트
- 체감: 점점 어두워지다가 일시정지 후 밝기 조정 모달이 뜬다
- 포인트: 친절해 보이는 UX가 결국 생존을 보장하지 않는다는 점

### 10. 안전지대 패턴

- 공개 라벨: `안전지대 패턴`
- 해금 기록: `END 09 여긴 안전합니다 (아님)`
- 의도: 분명히 안전해 보이는 통로로 유도한 뒤 그 통로만 닫아버리는 루트
- 체감: 정면에 길이 열리고, 한 번 닫힌 뒤 운 좋게 빠져나와도 곧바로 한 번 더 재계산이 들어온다
- 포인트: "여기 들어가면 되겠다"라는 판단 자체를 노리는 낚시 패턴

### 11. 대형 탄환 패턴

- 공개 라벨: `대형 탄환 패턴`
- 해금 기록: `END 10 대형 탄환`
- 의도: 처음엔 피할 수 있어 보이는 거대한 한 발이 결국 남은 공간까지 먹어치우는 루트
- 체감: 큰 탄환이 플레이어 높이로 들어오고 급속하게 커지며, 뒤늦게 바깥 탈출 경로까지 닫힌다
- 포인트: "옆으로 빠지면 되나?" 싶을 때 이미 늦어야 한다

### 12. 시간 단축 패턴

- 공개 라벨: `시간 단축 패턴`
- 해금 기록: `END 05 시간의 함정`
- 의도: 이번 판만 목표 시간이 30초로 줄어든 것처럼 보여 준 뒤 29초 부근에서 시간 오류로 끝내는 루트
- 체감: 초중반 난이도는 매우 쉬운 편이고, 거의 확실히 29초 근처까지 가게 만든 뒤 타이머가 그때부터 비정상적으로 꼬인다
- 포인트: 패배가 총알보다 시스템 오류처럼 보이게 하는 엔딩

## 도감 UI 규칙

- 잠긴 기록은 이름 대신 `???`로 표시된다.
- 해금된 기록은 카드 색이 바뀌고 `획득 완료` 배지가 붙는다.
- `END 99`는 잠겨 있어도 이름과 설명이 그대로 보인다.
- 수집 카운트는 `END 99`를 제외한 실제 해금 기록만 센다.
- 이미 보유한 엔딩은 다시 해금되지 않으며, 결과 화면에서도 중복 기록으로 표시된다.

## 사용 메모

- 테스트할 때는 도전 횟수에 따라 루트가 바뀌므로 순서를 알고 확인하는 편이 빠르다.
- 특정 패턴만 바로 보려면 저장된 도전 횟수나 기록 초기화 상태를 함께 확인해야 한다.
- UX 조정 시 가장 중요한 기준은 "처음엔 가능해 보이고, 나중에야 장난이라는 걸 깨닫는가"다.
