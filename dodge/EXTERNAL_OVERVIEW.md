# Zero Trace 소개 문서

`Zero Trace`는 60초 생존형 탄막 회피 게임이다. 플레이어는 포인터로 기체를 움직이며 밀려오는 패턴을 피하고, 반복 도전을 통해 기록 보관함을 채워 나간다.

## 특징

- 짧고 빠르게 반복 도전할 수 있는 60초 구조
- 플레이 기록이 누적되는 도감형 진행 방식
- 패턴이 점차 강해지며 분위기가 계속 변하는 구성
- 평범한 생존 게임처럼 시작하지만, 진행할수록 예상 밖의 연출이 등장하는 스타일

## 플레이 방식

- 마우스 또는 터치로 기체 이동
- 탄막 회피
- 반복 도전을 통한 기록 수집

## 구성 요소

- 생존 타이머
- 단계별 난도 상승
- 기록 보관함
- 여러 종류의 패턴 기반 엔딩

## 게임 톤

- 처음엔 익숙한 아케이드 게임처럼 보이도록 설계
- 뒤로 갈수록 분위기와 규칙 감각이 미묘하게 흔들리도록 구성
- 단순 난이도 상승이 아니라, 플레이 경험 자체가 변주되는 스타일

## 추천 포인트

- 짧은 플레이 안에서 강한 인상을 주는 구조
- 반복 도전 자체가 콘텐츠가 되는 방식
- 관전자와 플레이어 모두 반응하기 좋은 장면이 나오는 연출형 구성

## 현재 문서 범위

이 문서는 외부 공유용 요약 문서다. 상세 루트 구조, 도감 내부 조건, 숨겨진 연출 의도는 포함하지 않는다.
